📊 Personal Finance Manager

A full-stack Personal Finance Management System designed to help users track income, expenses, and get AI-powered financial advice.
Built with modern technologies including Spring Boot, React, JWT Security, and OpenAI API.

🚀 Features
🔐 Authentication & Security
User registration and login system
JWT-based authentication
Secure password encryption using BCrypt

👥 Role-Based Access Control
Two roles: USER and ADMIN
Protected endpoints using Spring Security
Admin-only APIs for restricted operations

💰 Financial Management
Track income and expenses
Calculate total income, expenses, and balance
Organized financial data for each user

🤖 AI Financial Advisor
Integrated with OpenAI API
Users can ask financial questions
Provides intelligent, real-time financial suggestions

📄 API Documentation
Swagger (OpenAPI) integration
Interactive API testing via browser

🧪 Testing
Unit testing for service layer
Integration testing for APIs
Ensures reliability and correctness

🐳 Containerization
Docker support for backend and frontend
Easy deployment using containers

Backend

Spring Boot

Spring Data JPA (Hibernate)

Frontend
